== ARCHITECTURE ==
start.sh -> MC server (background) + wait RCON -> Node.js panel (server.js)
server.js -> Express + Socket.IO + RCON client + Telegram bot
MC crashes -> start.sh restarts it in-process
Stop via STOPPED flag file (/data/STOPPED)
